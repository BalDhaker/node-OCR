`npm run start` to run ocr server
`npm run llm` to run ocr server with ollama and AI vision models,

install ollama desktop first,
start ollama if not running with:
`ollama serve`
then pull open source models with:
`ollama pull gemma3` 
or other models like `deepseek-ocr` etc
check list of models available at:
[Search Vision Models](https://ollama.com/search?c=vision)
